cf. html 태그라고 구글에 검색만해도 다양한 태그를 공부 할 수 있따.

<link>태그 - <head></head> 안에 작성
ex) <link href="" rel="stylesheet">
1. 외부의 css를 불러오는 태그
2. href는 어떤 링크의 파일을 불러올 것이냐
3. rel은 더 다양한 외부 폰트, 파일등을 불러올 수 있다. // stylesheet명시 해줘야 함.

<style> - <head></head> 안에 작성
1. 내부에서 직접 입력할수도 있다.

<script>태그 - 자바스크립트
ex) <script src=""></script>
1. 내부, 외부에서 자바스크립트 파일 불러올때 쓰인다.
2. src를 통해 가지고 온다. // 경로

<body>안에 들어감
1. 헤드라인태그 h1~6 까지있다 : 일반적으로 제목 점점 작아짐
<h1></h1>
<h2></h1>
<h3></h1>
<h4></h1>
<h5></h1>
<h6></h1>

2. <P> : 문단을 나눌때 사용(일반 택스트)

3. <br/> : 클로징태그 바로 달아줌. 일반적으로, 줄바꿈시킬때

3. <a> : 속성이 필요함 - href(필수), 링크텍스트

4. <img/> : 이미지를 보여줌 (클로징태그 바로) - 속성 : src(경로), alt도 넣어줌 : 만약 이미지나 이런거 안보일시 간략한 설명.

5. <form> : 프로세스를 돕는 태그, form안의 태그들을 관리를 해줌

6. <label> : 라벨을 붙임, ex)input태그에 제목을 붙인다.

7. <input> : 입력을 받음, 속성(type="text"  or "password", 다양한 타입이 많다)이 붙음.

8. <div> : 영역들을 나눌 때 사용한다. 하나의 박스로 생각하면 된다
