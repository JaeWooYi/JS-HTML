### State

컴포넌트는 return을 통해서 UI 요소를 보여준다. 
서비스를 사용하다 보면 컴포넌트의 UI를 업데이트 해줘야 하는 상황이 발생. 

1. 시간을 보여주는 화면을 구현할 때,시간을 표시하는 컴포넌트는 시간이 지나면서 계속 업데이트되는 것.
2. 네트워크 통신을 통해 데이터를 받아오면 화면 컴포넌트에 새로운 값을 보여주는 것.
3. 검색창에서 검색어를 입력할 때 아래에 추천 검색어를 바꿔서 보여주는 것.

**이때 컴포넌트를 업데이트(렌더링)해줄 수 있도록 해주는 장치가 바로 `State`.**  

**여기서 컴포넌트가 업데이트 된다는 것은 컴포넌트의 코드가 재실행되는 것을 의미.**

Props는 기본적으로 부모에게 값을 전달받는다.
전달받은 Props는 컴포넌트 내에서 따로 업데이트하지 않고 정적으로 사용. 

State는 Props와 반대로 변화하는(동적인) 값. 
기본적으로 state가 변경되면 컴포넌트는 새롭게 업데이트된다. 
(Props는 부모에게 받는 값, State는 컴포넌트 내에서 사용하는 값)

**state를 사용하기 위해선 React 라이브러리의 `useState`를 사용하면 된다.** 

1. useState 함수를 실행할 때 첫 번째 인자에 기본 값을 넣어준다.
2. useState함수를 실행하면 결과로 [state 값, state를 변경시키는 함수], 이렇게 2가지를 반환.
3. 좌측에 state 값을 **time 변수**를 넣고, state 변경 함수를 setTime 변수에 넣는다.
4. setTime 함수로 state를 업데이트 시키면 컴포넌트가 Re-Rendering(리 렌더링)된다.
